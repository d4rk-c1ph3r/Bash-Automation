
# Debian/Ubuntu (apt/dpkg)

```
sudo apt update -y
sudo apt-get install -y <package>
```

# Red Hat/CentOS (yum/rpm)

```
sudo yum update -y
sudo yum install -y <package>
```

# FreeBSD (pkg)

```
sudo pkg update -y
sudo pkg install -y <package>
```

# Build packages from source

```
git clone <repo>
./configure
make
sudo make install
```
